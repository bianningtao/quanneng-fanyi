关于

Transly 是本地优先的沉浸式网页翻译扩展。

当前能力网页双语翻译、文本翻译、划词、悬停、输入框、悬浮球、侧边栏和站点规则。
本地优先设置保存在浏览器存储；自定义 API Key 只保存在本机 local storage。
能力边界当前聚焦通用网页翻译和可见字幕翻译；移动端、文档、图片 OCR 不在本版范围内。
