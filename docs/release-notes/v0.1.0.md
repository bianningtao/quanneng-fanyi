# 全能翻译 v0.1.0

首个公开预览版，提供源码加载和 Release zip 两种安装方式。

## 安装

1. 下载 Release 附件 `quanneng-fanyi-v0.1.0.zip`。
2. 解压 zip。
3. 打开 Chrome，进入 `chrome://extensions/`。
4. 开启右上角“开发者模式”。
5. 点击“加载已解压的扩展程序”。
6. 选择解压后的 `all-in-one-translate-extension/` 文件夹。

## 主要功能

- 网页段落级双语翻译。
- 悬浮球快捷入口、侧边栏、弹窗和右键菜单。
- 划词翻译、悬停翻译、输入框翻译。
- 视频字幕翻译和字幕样式配置。
- Google、Microsoft、自定义 JSON、OpenAI-compatible 接口和演示服务。
- 可自定义翻译服务备份顺序。
- 术语库、自定义站点规则和多种译文样式。

## 已知限制

- 当前不是 Chrome Web Store 安装包，需要使用开发者模式加载。
- PDF、文档、图片 OCR、音频识别和字幕下载还未进入本版本。
- 自定义接口和 Microsoft 接口需要用户自行配置可用服务和密钥。
- 默认 Google 网页翻译会向 Google 翻译接口发送待翻译文本。
