# 全能翻译 v0.1.1

本版本补齐站点规则可视化、字幕下载和术语库真实生效链路，并继续强化翻译服务回退。

## 安装

1. 下载 Release 附件 `quanneng-fanyi-v0.1.1.zip`。
2. 解压 zip。
3. 打开 Chrome，进入 `chrome://extensions/`。
4. 开启右上角“开发者模式”。
5. 点击“加载已解压的扩展程序”。
6. 选择解压后的 `all-in-one-translate-extension/` 文件夹。

## 新增与改进

- 站点规则新增可视化编辑器：支持新增、删除、编辑、从 JSON 刷新和同步到 JSON。
- 站点规则字段覆盖域名匹配、包含选择器、排除选择器、保持原文选择器、最小文本长度和动态页面适配模式。
- 字幕文件翻译后可分别下载译文字幕和双语字幕，保留 SRT/VTT 时间轴。
- 术语库对免费引擎链路更有效：Google/Microsoft 请求前会保护匹配术语，自定义 API 继续接收术语约束。
- 翻译服务回退继续使用可配置顺序，跳过不可用服务并保留失败尝试信息。

## 已知限制

- 当前不是 Chrome Web Store 安装包，需要使用开发者模式加载。
- PDF 文本层、图片 OCR、音频识别仍不在本版本范围内。
- 实时视频字幕以播放时翻译显示为主；本版本下载能力主要面向 `.srt`、`.vtt` 文件。
- 自定义接口和 Microsoft 接口需要用户自行配置可用服务和密钥。
